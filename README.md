# ngrokformc
### My own alternative for whoever hosted mc server on ur pc but u cant port forwarding
### Region select
#### US (United States)
```
ngrok tcp -region us 25565
```
#### EU (Europe)
```
ngrok tcp -region eu 25565
```
#### AP (Asia Pacific)
```
ngrok tcp -region ap 25565
```
#### AU (Australia)
```
ngrok tcp -region au 25565
```
